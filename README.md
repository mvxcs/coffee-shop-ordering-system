# Coffee Shop Ordering System ☕

A real-time digital ordering and management system for cafés.

## Features
- Multi-language menu (Arabic / English / Turkish)
- Cart system with order review
- Order submission with invoice
- Live order status (Preparing / Ready)

## Dashboard
- Real-time incoming orders
- Mark orders as ready
- Order history
- Product control (Available / Sold Out)
- Notification for new orders

## Tech Stack
- HTML, CSS, JavaScript
- Supabase
